# Промты для генерации сцен — Vitgem Menopause (v3)

Эти промты предназначены для генерации **сцен БЕЗ банки** в ChatGPT / Nano Banana Pro / Midjourney. Дизайнер затем композитит банку (из `main_render.jpg`) поверх сцены и накладывает text overlay.

Промты намеренно оставлены **на английском** — генеративные модели работают лучше на английских промтах.
