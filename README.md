# Vitgem Menopause Gummies — Amazon UK Gallery Master Brief

This repository contains the interactive designer package, specs, and prompts for the Vitgem Menopause Gummies Amazon UK gallery redesign.

## Live Master Brief
You can view the interactive, beautifully styled version of this brief here:
👉 **[https://darkedid89.github.io/vitgem-menopause-gallery-brief/](https://darkedid89.github.io/vitgem-menopause-gallery-brief/)**

## What's Inside

- `/assets/`
  - `master_bottle_cut.png` — High-resolution transparent cutout of the master bottle (use this 100% as-is for compositing)
  - `logo_white.png` & `logo_dark.png` — Brand logos for light and dark backgrounds
  - `gummies_L.png` & `gummies_R.png` — Purple gummies assets for layout use
  - `color-swatches.svg` — Brand color swatch vector
  - `reference/` — Previews of the current JPG choices (`1.jpg` to `7.jpg`), alternative versions (`7-1.jpg`, `8-1.png`), and reference ChatGPT renders

- `styles.css` — Styling for the interactive HTML page
- `index.html` — The main designer dashboard
- `PROMPTS_FOR_USER.md` — Copy-pasteable prompts to generate backgrounds/scenes in ChatGPT
- `STYLE_AUDIT.md` — Visual analysis and guidance on keeping the slots cohesive
